Bienvenido a Pro Teleprompter Este es un teleprompter profesional para tus presentaciones y grabaciones de video. Características principales: • Control de velocidad ajustable • Modo espejo para reflectores • Cambio de tamaño de texto • Soporte para texto personalizado • Controles de teclado Instrucciones de uso: 1. Usa el botón "Editar Texto" para cargar tu guion 2. Ajusta la velocidad según tu ritmo de lectura 3. Activa el modo espejo si usas un reflector de teleprompter 4. Presiona ESPACIO o el botón Iniciar para comenzar 5. Usa las flechas arriba/abajo para ajustar velocidad en tiempo real Atajos de teclado: • ESPACIO: Reproducir/Pausar • R: Reiniciar • E: Editar texto • M: Modo espejo • F: Pantalla completa • ↑/↓: Ajustar velocidad ¡Comienza a grabar como un profesional!
