# ProTeleprompter

Un teleprompter profesional basado en web, diseñado para ser fluido, persistente y fácil de usar.

## Características

- 🚀 **Fluidez Total**: Animación basada en tiempo real (delta-time) para un desplazamiento suave en cualquier monitor.
- 💾 **Persistencia Automática**: Los ajustes y el guion se guardan automáticamente en el navegador (`localStorage`).
- 📂 **Soporte Word (.docx)**: Importa tus guiones directamente desde Microsoft Word o Google Docs (exportados a .docx).
- 🔄 **Modos de Espejo y Flip**: Ideal para usar con reflectores físicos de teleprompter.
- ⌨️ **Controles de Teclado**: 
  - `Espacio`: Play/Pause
  - `R`: Reiniciar
  - `↑ / ↓`: Ajustar velocidad
  - `M`: Modo Espejo
  - `E`: Abrir editor
  - `F`: Pantalla completa

## Cómo usar

Simplemente abre el archivo `teleprom.html` en cualquier navegador moderno.

## Tecnologías

- HTML5 / CSS3 (Vanilla)
- JavaScript (Vanilla)
- [Mammoth.js](https://github.com/mwilliamson/mammoth.js) (Para soporte .docx)
